To manage any of the following CRA security options, you must exit this service and then login.

 

- Change CRA user ID

- Change CRA password

- Change CRA security questions and answers

- Update additional security feature preference

- Revoke CRA user ID

- View the Terms and conditions of use

- View recent CRA login history


Once you have logged in, you can access these security options on the CRA login and security options page by selecting the Manage CRA security options button.
